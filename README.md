# 🕵️ Python Detective Game

A Python-based text adventure detective game where players solve criminal cases through evidence collection, suspect interrogation, clue analysis, and logical decision-making.

## Features
- Interactive detective gameplay
- Evidence collection
- Suspect interrogation
- Decision-based story progression
- Command-line interface

## Technologies Used
- Python
- Visual Studio Code

## How to Run

1. Clone this repository.
2. Open the project in Visual Studio Code.
3. Run:

```bash
python your_file_name.py
```
